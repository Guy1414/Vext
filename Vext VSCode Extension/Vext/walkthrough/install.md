# Compiler Check
Vext needs a compiler binary to provide language support and run your code.

### ⚡ Status Bar
Look at the bottom right of your VS Code window. You should see a **lightning bolt icon** with the current Vext version. 
- If it's there, the compiler is ready!
- If it says "Starting...", we're fetching the latest binary for you.

[Check for Updates...](command:vext.forceUpdateCheck)
