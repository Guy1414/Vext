# Your First Run
Running Vext code is simple and integrated directly into the editor.

### 🚀 How to Run
1. Open any `.vext` or `.vxt` file.
2. Click the **Play button** in the top-right corner of the editor.
3. Your output will appear in the dedicated **Vext Run** terminal at the bottom.

### 🔬 Problems View
If there are errors in your code, Vext will underline them. You can't run a file with critical errors—consistency is key!

[Run active file](command:vext.runCode)
